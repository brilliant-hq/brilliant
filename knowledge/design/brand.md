# Design Brand DNA

## Logo Variety

Default AI logo — icon in solid box — is generic. Break the pattern:

| Logo Style | Best for |
|---|---|
| **Naked icon** (no background) | Confident, minimal |
| **Outlined circle** | Premium SaaS, fintech |
| **Gradient mark** | Modern SaaS, AI/tech |
| **Subtle fill** (10% opacity bg) | Dashboards, tools |
| **Icon + accent shape** | Startups, playful |
| Solid filled box | Only when brand actually uses this |

## Threading Brand Language

Logo's visual DNA should echo throughout. Pick 2-3 echo points:

| Logo trait | Echo in |
|---|---|
| Circular `rd(9999)` | CTA buttons, avatar frames, badges |
| Rounded square `rd(8)` | Cards, feature icon boxes, inputs |
| Gradient fill | Accent bar, hero headline |
| Outline/stroke style | Cards use stroke not fill |
| 10% opacity fill | Feature icon boxes, tag backgrounds |
| Accent shape (dot, circle) | Section dividers, bullets |

**The test:** Cover the logo. Can you still guess the brand's personality?
